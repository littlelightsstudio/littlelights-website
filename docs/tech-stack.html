Ziel-Registrar

Pilot läuft: nocturnedomains.com Transfer initiiert 27.04.2026, in „being processed". 16 Domains insgesamt nach Migration.

Wichtig zur Hetzner-Domain-Verwaltung

Hetzner hat keinen Standalone-Domain-Service à la Cloudflare/Porkbun. Domain-Registrierung ohne Webhosting läuft über das konsoleH-Panel, Produkt „Domains → Registration only". Bestellt wird via accounts.hetzner.com, das Panel-Login danach ist konsoleH.

Zukünftige Migrationen (Reihenfolge)

  1. Phase 1 (DNS-Cutover): Alle 16 Domains von easyname-NS auf Hetzner-NS umstellen — in Arbeit
  2. Coolify-Apps-Setup: helenaflinn.com → michaelsokolar.com → littlelights.studio → ferienwohnung-mistelbach.at als Coolify-Container
  3. Phase 3 (Redirect-Migration): 8 Brand-Schutz-Domains' Redirects von easyname auf einen eigenen Caddy-Container in Coolify. Separates Repo littlelights-redirects mit Caddyfile als Source of Truth. Pflege liegt bei miso, Lukas baut nur Tools. Details in roadmap.md.
  4. Phase 4a (Email-Migration): sokolar.com + sokolar.at + helenaflinn.com Email zu Forward Email
  5. Phase 4b (Registrar-Transfer): Domain-für-Domain easyname → Hetzner Domains. Pilot: nocturnedomains.com (27.04.2026 in Arbeit). nocturnecodex.com nachziehen sobald 60-Tage-Lock fällt.
